<summary><b>⚙️ How to activate the snake animation above (one-time, ~2 min)</b></summary>
<br>

1. In this repo, go to **Settings → Secrets and variables → Actions** — nothing needed here, `GITHUB_TOKEN` is automatic.
2. Create `.github/workflows/snake.yml` with:

```yaml
name: Generate Snake
on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:
  push:
    branches: [ main ]

permissions:
  contents: write

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        id: snake
        with:
          github_user_name: dharmik-21
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=dracula
      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

3. Push it, then run the workflow once manually from the **Actions** tab.
4. It creates an `output` branch holding the SVG — the `<img>` tag already in this README points at it, so no further edits needed.

</details>
