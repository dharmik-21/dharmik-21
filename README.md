## Selected work

### Cortex Lens
**A second set of eyes, designed with care.**

An assistive smart-glasses system for people with visual impairments. It detects nearby objects, estimates distance, reads printed text, and responds with immediate audio feedback. Built around a Raspberry Pi, camera, tactile input, and a deliberate “smart-zone” safety system.

`Python` · `YOLOv8` · `PyTorch` · `MiDaS` · `OpenCV` · `EasyOCR` · `Raspberry Pi`

<br />

### Hisab AI
**Financial data that talks like a teammate.**

An invoice intelligence system that extracts data from uploaded bills, organizes GST information, visualizes business performance, and answers questions such as “What was my top product this month?” in natural language.

`Streamlit` · `Tesseract OCR` · `Pandas` · `Plotly` · `MySQL` · `NLP`

<br />

### CycloneVision AI
**Finding rare patterns in satellite imagery.**

A one-class cyclone detection project using reconstruction error, CNN-based localization, and a real-time interactive dashboard.

`TensorFlow` · `Autoencoders` · `CNN` · `OpenCV` · `NumPy`

<br />

### Blog Generator AI
**A more intentional way to create with LLMs.**

A content generation tool that combines NLP pipelines, transformer models, and LangChain orchestration to create articles with controllable tone, structure, and purpose.

`LangChain` · `LLMs` · `Transformers` · `Scikit-learn` · `Streamlit`
