I'm an **AI/ML Engineer** and MCA (AI & ML) student at Jain University, specializing in Large
Language Models, Agentic AI, and Computer Vision. Right now I'm building **CortexLens**, wearable
smart glasses that give real-time spatial awareness to the visually impaired. I build with
TensorFlow, PyTorch, LangChain, Hugging Face Transformers, and the OpenAI API — turning research
ideas into working, data-driven products.

I've also shipped **HisabAI** (invoice processing & financial analytics), **CycloneVisionAI**
(satellite-based cyclone detection), and an **AI Blog Generator** built on LangChain. Alongside
deep learning, I'm comfortable across the **MERN stack** (MongoDB, Express, React, Node.js) and
FastAPI, so I can take a model from notebook to a real, usable app.

📫 Reach me at **dharmik.aieng@gmail.com** · 📍 Bangalore, India
